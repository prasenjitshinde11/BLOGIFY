📝 Blogify
A full-featured content management system built with Python and Flask.
Blogify is a robust web application that allows users to register, create, manage, and share content. It features a secure authentication system, dynamic content rendering, and a modular architecture using Flask Blueprints for scalability.
🚀 Key Features
User Authentication: Secure registration and login system with password hashing.  
CRUD Functionality: Users can Create, Read, Update, and Delete their own posts.
Profile Management: Custom user profiles with profile picture uploads and account updates.
Password Recovery: Email-based password reset functionality.  
Categorization: View posts filtered by specific users.
Responsive Design: Styled with Bootstrap for a seamless mobile and desktop experience.  
🛠️ Tech Stack
Backend: Python, Flask (Blueprints, Flask-Login, Flask-Mail)
Database: SQLite / SQLAlchemy (with Flask-Migrate)
Frontend: HTML5, CSS3, Bootstrap 5, Jinja2 Templating
Version Control: Git & GitHub
